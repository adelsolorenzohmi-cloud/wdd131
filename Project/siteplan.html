Routine Flow Site Plan.

1. Site Name and Idea

Site Name: Routine Flow

the name is easy. It means thing you do every day should be smooth and simple.

What is the site? This website is a To-Do list for good habits. It helps a person check off important things they do daily, like reading or working.

2. Site Purpose (What the side does)

The site is a simple tool to help you track your daily tasks. It helps you start good habits. You can see when you finish a task, and the list stays saved in your browser so you don't lose it.

The site has three pages.

  • Main Page (active tasks). Here, you can add new tasks. You can click a button to say a task is Done.
  • Goals Page. This page will show your goals for the whole week.
  • Contact Page. This page has a form where you can write a message to the site creator.

3. Scenarios (Questions from a Visitor)

These are questions a person might ask when using the site:

  • Question 1: I need to add a new task, like "Study for 1 hour." How can I write the task and choose a category like 'Work' quickly?
  • Question 2: I only want to see my 'Habit' tasks, not my 'Work' tasks. Where is the filter button to show only habits?
  • Question 3: I finished my task. When I click the 'Done' button, how does the list show me that the task is finished (like changing its look)?

4. Color Schema (The site's colors)

The colors are for calm, success, and good visibility.

Color Name Color (Simple) Use on the Site
Primary Green (#4CAF50) For buttons that mean success (like "Done") and main titles.
Secondary Blue (#2196F3) For other buttons (like the filters).
Background Light White (#FAFAFA) For the background of the entire page.
Text Dark Gray (#424242) For all the words on the page.

5. Typography (The site's fonts)

We use two types of fonts to make the site clean and easy to read.

  • Big Titles (H1, H2): We use a strong font like 'Open Sans'.
  • Regular Words (Paragraphs, List): We use a simple font like 'Arial'.

6. Wireframe Layout (The site's simple drawing)

This is how the main page will look on different screen sizes.

Mobile Phone View

Everything is in one long line. The header, the form to add tasks, the filter buttons, and the task list all stack one on top of the other.

Big Screen View (Desktop)

The screen is split. The form to add tasks is on the left side, and the task list and filter buttons are on the right side.
